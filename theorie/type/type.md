<h1>Les types</h1>
<section>
   <p>
      Pour chaque les variables fonctionnnent correcetement, elles ont besoin des typages. Les types permets de determiner quel natures est la variable. Ils sont au nombre de 4 : <br>
 <div>
   <ul>
     <li><strong>Chaine de caractère(str)</strong></li> Serve a assigné une variable a une type de chaine c'est a dire des lettre avec les doubles quote "" ou meme single quote ''.  <br> Remarque:
     <mark>Si on a "123456", le programme le considere comme chaine de caractere.</mark> . <br>
     <code>nom = "John Doe"</code> ou
      <code>nom = 'John Doe'</code> <br><br><br> 
       <li><strong>Entier(int & float)</strong></li> Serve a assigné une variable a une type d'entier ou numerique. On e dispose 2 avec int et float <br>
     <code>age = 123456</code> &
      <code>moyenne = 12.9</code> <br><br><br> 
      <li><strong>Boolean(bool)</strong></li> Serve a assigné une variable a une type booleene. Important pour tous ce qui est la verification avec les mots True or False<br>
     <code>isStudent = True</code> &
      <code>isRain = False</code> 

   </ul>
 </div>
